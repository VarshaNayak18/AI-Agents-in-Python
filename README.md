# AI Agents in Python

This repository contains implementations of different types of Artificial Intelligence agents using Python.  
The goal of this project is to understand how intelligent agents perceive their environment, make decisions, and take actions.

These agents demonstrate key AI concepts such as reactive behavior, goal-driven decision making, planning, learning, and pathfinding.

## Implemented Agents

### 1. Simple-Reactive Agent
A simple rule-based agent that reacts directly to the environment using conditional logic.

### 2. Goal-Based Agent
This agent selects actions based on a predefined goal and evaluates possible states to achieve that goal.

### 3. Utility-Based Agent
Chooses the best possible action by evaluating a utility function.

### 4. Memory-Based Agent
Stores previous inputs or states and uses them to influence future decisions.

### 5. Model-Based Agent
Maintains an internal model of the environment to make better decisions.

### 6. Learning Agent
Improves its behavior over time using feedback from previous interactions.

### 7. Planning Agent
Creates a sequence of actions to achieve a goal.

### 8. Path Finding Agent
Implements a pathfinding strategy to move from a start point to a goal state.

### 9. Game Playing Agent
Demonstrates decision-making in a simple game environment.

### 10. Recommendation Agent
An agent that suggests items based on preferences.
